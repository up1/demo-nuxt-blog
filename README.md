# Demo Blog app with NuxtJS
* NuxtJS
* Tailwind
* State management with Pinia
* Working with API (axios) + OpenAPI
* Testing
* Working with Docker

## Run project

Run
```
npm run dev
```

Upgrade version
```
npx nuxi upgrade --force
```

Clear cached
```
npx nuxi cleanup .
```

## Generate type from OpenAPI
```
npx openapi-typescript ./postman/openapi.yml -o ./openapi.gen.ts
```
